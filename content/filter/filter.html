1.定义

  • 过滤器用来格式化数据以更加友好的方式展示给用户。
  • 过滤器可细分两类，一种是Angular内置的过滤器，一种是用户自定义的过滤器；
Angular内置过滤器
# 过滤器 可否传参 描述
1 currency Yes 格式化数字为货币格式。其实就是向数字钱添加一个字符串；
2 date Yes 将日期格式化成需要的格式 。
3 filter Yes 从数组项中选择一个子集。
4 json No 将一个JSON或JavaScript对象转换成字符串。这种转换对调试非常有帮助。
5 limitTo Yes
  • 根据传入的参数生成一个新的数组或字符串。
  • 新的数组或字符串的长度取决于传入的参数。
  • 通过传入参数的正负值来控制从前面还是从后面开始截取。
6 number Yes number过滤器将数字格式化成文本。它的第二个参数是可选的，用来控制小数点后截取的位数。如果传入了一个非数字字符，会返会空字符串。
7 orderBy Yes 根据某个表达式排列数组。
8 lowercase No 格式化字符串为小写。
9 uppercase No 格式化字符串为大写。

2.内置过滤器的使用

使用管道符“|”后面添加过滤器的名字即可使用，给过滤的传递参数是在过滤器的后面加“:”(英文冒号)，如果传递多个参数请使用多个英文冒号隔开，然后在后面加引号，在引号里面写上参数；

示例： 
<p>{{ employees|orderBy:"salary"|json }}</p>
或
ng-repeat="e in employees|filter:{'name':'!'+uname}|orderBy:'salary'"

3.自定义过滤器

本质：过滤器本质上是一个会把我们输入的内容当作参数传入进去的函数 ； 步骤：
  • 创建一个模块，取得模块对象；
  • 在模块对象上调用 filter 方法，第一个参数是自定义过滤器的名字，第二个参数是一个函数，可以给该函数注入一些服务以便内部使用，最后该函数返回一个函数，并且返回的的这个函数也要有一个返回值；
示例： 
//删除非字母字符过滤器
app.filter("removeNotLetter",function(){
	return function(input){
		if(input){
			return input.replace(/[^a-z]/ig,'');
		};
	};
});
